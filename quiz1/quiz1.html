Intro ITWS

Create a personal Intro to IT website in your RCS iit directory

  • The web site will be placed in an "iit" subdirectory of your RCS file space, and it must be RCS password protected: you will give rights to Megha and me. Follow these instructions to set up your protected directory.
  • Guidelines:
    • You must have at least two pages: home and projects. Name these files:
      • index.html (this will be your landing page, or home page)
      • projects.html (this will be the main page to introduce your labs)
    • You must have a consistent header, menu, content area, and footer.
    • The only file in your root (iit) folder (at the root of your site) is the index.html page
    • All resources (e.g. CSS files) should be kept in a separate area(s) as per your design.
    • Populate your projects page with links to the labs, beginning with your resume; continue to do this as we move forward this semester.
    • As before, your code must be valid and properly indented. (And don't forget to set the <title> tag on each page.)
  • We will work through some of this together in class.
  • You will graded using the Lab 3 rubric as a guideline. Creativity will be rewarded.
  • I have included an HTML template to help you get started: HTML5 template
  • Feel free to experiment and be creative - and use the web as a reference!! (e.g. W3Schools)
